# 关于单仓项目 vscode i18n-ally 插件无法识别问题

## 问题描述

在单仓项目中，使用 vscode i18n-ally 插件时，无法识别项目中的多语言文件。

## 解决方案

在项目根目录中创建一个 `.vscode/settings.json` 文件，添加如下配置：

```json
{
  // 单仓项目需要主动声明所使用的框架。
  "i18n-ally.enabledFrameworks": ["vue"]
}
```

## 需要注意的条件

* Vue-i18n 必须使用 `vue-i18n@8.0.0` 以上版本。
* Typescript 必须使用 `typescript@5.0.0` 以上版本。
